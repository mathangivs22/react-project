# ReactProject
Beginner friendly todo app uisng React. 
This is a Todo App to keep track of the tasks we have and the number of items completed successful.
It uses basic React concepts like components ,states etc.

# Features
- Add new tasks
- Delete unwanted tasks
- Click on the task if it has been completed.

# Concepts Used
- React Components
- State management (useState)
- Props
- Event Handling

# Tech Stack
- React
- Javascript
- HTML
- CSS
